# Taskify – Kundenhandbuch

## Anmelden

Rufen Sie die vom Support-Team bereitgestellte Portal-URL auf und klicken Sie auf **Kunden-Login**. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein.

---

## Ticket einreichen

1. Klicken Sie auf Ihrem Dashboard auf **Neues Ticket** (oder füllen Sie das Formular auf der Startseite aus, wenn Sie kein Konto haben).
2. Geben Sie einen **Betreff** ein und beschreiben Sie Ihr Anliegen.
3. Gehören Sie mehreren Projekten an, wählen Sie das zutreffende **Projekt** aus der Auswahlliste.
4. Klicken Sie auf **Absenden**.

Sie erhalten eine Bestätigungs-E-Mail mit einem Link zur Nachverfolgung Ihres Tickets.

---

## Ihr Ticket verfolgen

Melden Sie sich an und klicken Sie auf den Ticketbetreff. Sie können auch den Tracking-Link aus Ihrer Bestätigungs-E-Mail verwenden – ohne Anmeldung.

---

## Antworten

Scrollen Sie zum Ende des Tickets, geben Sie Ihre Nachricht ein und klicken Sie auf **Antwort senden**. Ihr Support-Team wird automatisch benachrichtigt.

---

## Ticket-Status

| Status | Bedeutung |
|--------|-----------|
| Offen | Eingegangen, noch nicht gestartet |
| In Bearbeitung | Wird bearbeitet |
| Gelöst | Behoben – antworten Sie, wenn Sie weitere Hilfe benötigen |
| Geschlossen | Keine weiteren Maßnahmen erwartet |

---

## Projekttickets

Wenn Sie Mitglied eines oder mehrerer Projekte sind, erscheint auf Ihrem Dashboard der Tab **Projekttickets**. Dort sehen Sie alle Tickets, die Ihren Projekten zugewiesen sind – auch Tickets anderer Mitglieder dieser Projekte.

---

## Probleme mit Ihrem Konto?

Wenden Sie sich an Ihr Support-Team – es kann Ihr Passwort zurücksetzen oder Ihr Konto reaktivieren.
